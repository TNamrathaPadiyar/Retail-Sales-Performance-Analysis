# Retail-Sales-Performance-Analysis
This project analyzes retail sales performance using Power BI to evaluate total sales, profit, category contribution, and regional activity. The analysis highlights strong revenue, balanced category performance, and opportunities for regional expansion based on sales patterns.


## Key Findings
- Total Sales: 24.25M  
- Total Profit: 3.69M  
- Total Quantity Sold: 4948 units  
- Profit Margin: 15%  
- Sales distribution is balanced across categories.  
- Regional performance is uneven, showing expansion opportunities.
